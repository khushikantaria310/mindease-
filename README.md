 🌿 MindEase+

An aesthetic mental wellness web app designed to help users track mood, relax, and reflect.

 🚀 Features

- 🧠 Mood detection from daily input  
- 🔋 Mood battery visualization  
- 🎧 Mood-based video playlist  
- 💬 Affirmation generator  
- 📝 Personal journal (saved locally)  
- 🎨 Modern aesthetic UI  


 🛠️ Tech Stack

- HTML  
- Tailwind CSS  
- JavaScript  
- Char.js
